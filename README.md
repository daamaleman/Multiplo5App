# Multiplo5App - Verificador Pro 🔢

![Android](https://img.shields.io/badge/Platform-Android-brightgreen.svg)
![Kotlin](https://img.shields.io/badge/Language-Kotlin-orange.svg)
![Jetpack Compose](https://img.shields.io/badge/UI-Jetpack%20Compose-blue.svg)
![Material 3](https://img.shields.io/badge/Design-Material%203-purple.svg)

Una aplicación Android moderna y elegante diseñada para verificar si un número es múltiplo de 5 y generar tablas de multiplicar interactivas. Construida con las últimas tecnologías de desarrollo nativo.

## 🚀 Características

- **Verificación Inteligente:** Determina instantáneamente si un número es múltiplo de 5.
- **Tablas Interactivas:** Si el número es válido, se despliega una tabla de multiplicar detallada (1-12) en un modal elegante.
- **Interfaz Moderna:** Diseño basado en **Material 3** con soporte para temas dinámicos.
- **Experiencia de Usuario:** Validaciones de entrada en tiempo real y animaciones suaves.
- **Responsive:** Adaptable a diferentes tamaños de pantalla mediante componentes de Compose.


## 🛠️ Tecnologías Utilizadas

- **Kotlin:** Lenguaje de programación principal.
- **Jetpack Compose:** Toolkit moderno para construir UIs nativas.
- **Material 3:** Sistema de diseño de última generación.
- **Kotlin Coroutines:** Para operaciones asíncronas fluidas.
- **Android Architecture Components:** Siguiendo las mejores prácticas de Google.

## 📥 Instalación

Para probar este proyecto localmente, sigue estos pasos:

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/tu-usuario/Multiplo5app.git
   ```
2. **Abre el proyecto:**
   Importa la carpeta en **Android Studio (Ladybug o superior)**.
3. **Sincroniza Gradle:**
   Espera a que termine la descarga de dependencias.
4. **Ejecuta la app:**
   Selecciona un emulador o dispositivo físico y presiona `Run`.
